■導入方法
Eclipseで実行する場合、ファイル -> 新規 -> プロジェクトをクリックします。
プロジェクトウィザードが立ちあがるので、JAVAプロジェクトを選択します。
任意のプロジェクト名を記述した後、外部ソースからプロジェクトファイルを選択をクリックします。
ディレクトリーにjist-swansフォルダへのパスを記述し、次へをクリックします。
デフォルト出力フォルダーがjist-swans/binとなっているのを確認し、終了をクリックします。


■更新方法
** 念のため旧ファイルのバックアップを取ることを強く推奨します **

jist_swansフォルダにあるsrcを上書きしてください。
自身で改変を加えたファイルがある場合は、変更箇所を確認して導入してください。
